# Enterprise Active Directory & SOC Home Lab

## Overview
This project demonstrates the design, deployment, and security monitoring of a small enterprise Active Directory environment. The lab simulates real-world infrastructure, user behavior, and attack scenarios to practice IT administration and cybersecurity detection skills.

## Objectives
- Deploy Windows Server 2022 as a Domain Controller
- Configure Active Directory, DNS, and Group Policy
- Join Windows clients to the domain
- Centralize log collection and monitoring
- Simulate attacks and analyze security events

## Technology Stack
- Windows Server 2022 (Active Directory, DNS)
- Windows 10/11 (Domain-joined client)
- Ubuntu Server 22.04 (SIEM / log collection)
- Kali Linux (attack simulation)
- VMware Workstation Player

## Key Skills Demonstrated
- Active Directory administration
- Windows & Linux system management
- Network segmentation and isolation
- Log analysis and threat detection
- Troubleshooting and documentation
